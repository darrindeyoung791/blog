    def bubble_sort(items):
    for i in range(len(items)):
        for j in range(len(items) - 1 - i):
            if items[j] > items[j + 1]:
                items[j], items[j + 1] = items[j + 1], items[j]

    4. 表格示例

    名称 图标 优点 缺点
    Microsoft Edge 与系统高度结合；功能强大易用；可以很方便地安装浏览器插件；手机电脑多端同步；很适合 PDF 阅读和批注。 臃肿；系统软件无法卸载。
    Mozilla Firefox
    （火狐浏览器）    		 自由开源浏览器，快速、安全、保护隐私，可以很方便地安装浏览器插件。 由于法律原因，国内无法在 Firefox 上安装广告拦截插件；有的网页不适配 Firefox；设置多而复杂；没有内置的网页翻译。
    Google Chrome
    （谷歌浏览器）    		 浏览器的代名词；（在国内大部分地方）纯粹快速的浏览器；外观现代简洁；快速。 由于不知道的原因，国内大部分地方在 Chrome 上安装插件和使用网页翻译需要额外付出努力；以及 Chrome 是被仿冒最多的浏览器之一，许多打着「AI」和「双核」浏览器的名号的仿冒浏览器会诱导下载，找到官网下载地址比较困难。
