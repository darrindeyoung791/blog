# OpenCode 也限制了免费方案……

今早四点刚过，OpenCode 突然提示已到达免费限度，建议购买 OpenCode Go。

![界面还挺炫酷的](/img/PixPin_2026-05-25_06-14-41.webp)

没有非常意外，只是觉得手头工作被打断，有点不愉快。

无论是谁，能无限制免费到现在，已经很了不起了。
